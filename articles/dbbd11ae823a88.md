# Laravel/PHPの環境(2023/5/13)
- Laravel Framework 8.83.27
- PHP 8.2.5
# コメントの書き方
```blade.php
{{-- <x-tests.app> --}}
```
のように`{{--  --}}`で囲むとOK。
**HTMLなどのコメントはコードだと思って認識されてしまうので注意**

# bladeファイルの書き方
- 始まりと終わりを以下のように囲む
```
<x-tests.app>
  component
</x-tests.app>
```

# 今回のミスの原因
### ⚠️コメントの書き方がまずかった⚠️
- まずかったポイント↓(Before)
```
<!-- コメントなんか書いてた -->
<x-tests.app>
  component
</x-tests.app>
```
エラーが出る、、、なんで？
- 修正後↓(After)
```
{{-- コメントの正しい形式 --}}
<x-tests.app>
  component
<x-tests.app />
```
`<!-- コメントなんか書いてた -->`をコメントアウトしたら治った。**コメントの形式には注意。**

# 結論
#### コメントの書き方は`{{-- コメント --}}`である。
